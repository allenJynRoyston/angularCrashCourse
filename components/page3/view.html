ng-repeat With Simple Arrays

Using ng-repeat in it's most basic form requires two parts: an array to iterate through ($scope.basicArray) and a variable to refer to each iteration (in this case, entry). Note that we can name our iteration variable anything we want. We could call it something else and it would still have the same effect (with some restrictions). You should also note that when we refer to a $scope object in the view, we drop the $scope prefix.

HTML:

{{arrayDisplay_ul}}

Javascript:

$scope.basicArray = ["Mark", "Henry", "Steve", "Jimmy"];

Results:


	
{{entry}}
